【大列表】常用PC软件-2021年3月15日版本

本文最后更新于：2021年3月15日 下午

前言

本软件列表都是一些我用过的软件，很多都有相似功能的软件，我的倾向是如果功能类似，则优先选择使用开源免费软件。

　

　

程序开发

推荐

软件名称 官方网址 验证时间 适用场景 备注说明
Python https://www.python.org 2020-10-15，可用 运行 Python 代码 开源，官方软件
Mingw-w64 http://mingw-w64.org 2020-10-15，可用 在 Windows 上编译 C/C++ 开源，GCC 的64位 Windows 版本
Racket https://racket-lang.org 2020-10-15，可用 运行 Scheme 代码 开源，教学用软件
SBCL http://www.sbcl.org 2020-10-15，可用 运行 Common Lisp 代码 开源，仍然在频繁更新的解释器
flat assembler http://flatassembler.net 2020-10-15，可用 编译 汇编代码 开源，汇编工具
Java SE https://www.oracle.com/java/technologies/javase-downloads.html 2020-10-15，可用 编译 Java 代码 官方软件
Node.js https://nodejs.org 2021-3-15，可用 运行 Hexo 开源，本网站正在使用

不推荐

软件名称 官方网址 验证时间 适用场景 备注说明
TCC https://bellard.org/tcc 2020-10-15，可用 在 Windows 上编译 C/C++ 开源，功能较少，很久没有更新了

　

文件管理

软件名称 官方网址 验证时间 适用场景 备注说明

WinMerge https://winmerge.org 2020-10-15，可用 可对比两个文件的差异 开源，可以对比三个文件
7-Zip https://www.7-zip.org 2020-10-15，可用 可压缩、解压文件 开源，使用简洁
Syncthing https://syncthing.net 2020-10-15，可用 可在不同设备之间同步文件 开源，支持多平台
Free Video Compressor http://www.freevideocompressor.com 2020-10-15，可用 可压缩视频文件 开源，操作简单
