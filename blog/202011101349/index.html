2020年第4次公告：本站「图片库」已升级，图片载入更快

本文最后更新于 2022年3月28日 晚上

　

本站一直使用自建的「图片库」储存文章中的图片，但因为图片库是放在 github 上，导致在国内访问有可能速度缓慢。

所以这次对「图片库」进行了升级，将其托管在 Vercel 上。

Vercel 在国内访问速度也很不错，因为它自带 CDN 。

升级过程中，遇到了很多问题。例如，无法登陆 Vercel，通过给工程师发邮件解决了。

这里推荐一下 DeepL 这个翻译软件，和 Vercel 工程师沟通全靠它了。

成功注册登录之后，经过几个简单的操作，将 Vercel 与 github 成功关联到了一起，这样 github 更新图片后，Vercel 也会自动更新。

最后，在设置一下域名解析，将域名指向 Vercel，就完成了全部升级操作。

理论来说，现在图片的载入速度应该快于之前，如果出现异常，我之后再解决。
